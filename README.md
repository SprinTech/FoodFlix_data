# FoodFlix_data

## Context
FoodFLix is a Foodtech startup which currently work to a new application that give recommendation of products according to the nutri-score. Our goal is to create the MVP of this application by using an existing dataset.

## FIles
- Dataset URL : https://www.kaggle.com/openfoodfacts/world-food-facts
- Data_exploratory : exploratory analysis of Dataframe (shape, nb columns, comments about cleaning)
- Data_cleaning : cleaning of dataset
